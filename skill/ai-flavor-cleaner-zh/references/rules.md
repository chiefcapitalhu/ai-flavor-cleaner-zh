# 中文 AI 味检查与改写规则

## 使用方式

先判断表达在当前上下文中的作用，再选择处理动作。A 类通常可以明确处理；B 类只提示检查；C 类依赖全文结构和语义。

统一处理顺序：先删无信息内容，再核实重要信息，然后改成原文已有的对象、动作和结果。禁止近义套话替换，保留真实的不确定性。

## A. 明确句式问题

| 编号 | 模式 | 常见问题 | 优先动作 |
|---|---|---|---|
| A01 | 不是……而是…… | 制造假反转 | 直接说后半句；两边都有信息时改成真实关系 |
| A02 | 不仅……而且/更是…… | 强行拔高 | 保留事实，删除拔高 |
| A03 | 不再只是……而是…… | 用句式代替变化证据 | 写清新增、减少或改变了什么 |
| A04 | 真正关键的是 / 真正的核心是 | 用强调代替判断 | 删除强调词，直接说判断 |
| A05 | 本质上 | 跳过论证宣布本质 | 改成观察、因果或明确的个人判断 |
| A06 | 这意味着 | 给事实追加空泛解释 | 有新增信息就直接说结果，否则删除 |
| A07 | 从 A 到 B | 两端未必构成真实尺度 | 写清对象、时间和状态变化 |
| A08 | 如果说 A 是，那么 B 就是 | 比喻架子拖慢表达 | 直接说明两者关系 |
| A09 | 一方面……另一方面…… | 论文式平衡 | 先说影响结论的一面 |
| A10 | 总而言之 / 综上所述 | 作文式重复收尾 | 直接结束或只留行动 |
| A11 | 值得注意/一提的是 | 提醒本身没有信息 | 直接说后面的事实 |
| A12 | 我们可以发现 / 不难发现 | 替读者宣布结论 | 给事实或明确说是作者判断 |
| A13 | 接下来 / 先说 / 最后 | 暴露提纲和培训腔 | 用问题、动作或自然承接推进 |
| A14 | 今天我们学习 / 下面教大家 | 把内容变成课程开场 | 直接进入使用场景或问题 |
| A15 | 有一个简单的方法 / 正确做法是 | “简单/正确”缺少依据 | 直接展示动作和结果 |
| A16 | 建议大家 / 你需要做的是 | 居高临下 | 改成具体场景中的可选动作 |
| A17 | 没有套路、没有废话、没有门槛 | 连续否定制造宣传感 | 删除保证，直接交付 |
| A18 | 你早就/心里已经知道答案 | 制造虚假共鸣 | 换成真实问题或选择 |
| A19 | 停下来感受 / 花时间消化 | 模仿演讲停顿 | 删除，让事实建立重量 |
| A20 | 这就是全部/一切的核心 | 绝对判断制造金句 | 写清适用范围或删除 |

## B. 高风险表达

命中不等于错误。检查它是否提供了可核实的对象、动作、条件或结果。

| 类型 | 代表表达 | 检查问题 | 处理方向 |
|---|---|---|---|
| 抽象名词 | 范式、体系、能力、生态、价值、闭环、底层逻辑 | 具体指什么？谁做了什么？ | 使用原文已有的对象和动作 |
| 空泛预测 | 未来、方向、升级 | 时间、对象和依据是什么？ | 写清已经发生的变化 |
| 意义拔高 | 革命、重构、重塑、重新定义 | 是否有可验证的前后差异？ | 改成前后对比或删除 |
| 新闻稿腔 | 标志着、见证了、体现、彰显、象征 | 是否只给事实追加意义？ | 保留事实和有来源的解释 |
| 重要性宣告 | 至关重要、关键转折点、凸显重要性 | 不处理会怎样？ | 写清影响对象和后果 |
| 模糊未来 | 为……奠定基础、不断演变的格局 | 已经具备了什么条件？ | 只说已发生的事实 |
| 宣传形容词 | 深刻、宝贵、丰富、充满活力、开创性、叹为观止 | 删除后信息是否减少？ | 用可观察细节替代 |
| 空泛动词 | 增强、促进、培养、赋能、展示、强调、突出 | 到底发生了什么？ | 改成功能、行为或结果 |
| 机械连接 | 此外、然而、与此同时 | 是否真的存在递进或转折？ | 能直接接就删除 |
| 模糊合理性 | 与……保持一致 | 标准、文件或要求是什么？ | 指明具体依据 |
| 匿名权威 | 行业报告显示、专家认为、观察者指出、多个来源表明 | 谁、何时、在哪里说的？ | 补具体来源，否则删除归因 |
| AI 免责声明 | 基于可用信息、虽然具体信息有限 | 未知边界到底在哪里？ | 标明未知并停止推测 |
| 过度限定 | 可以潜在地可能、可能会产生一些影响 | 哪一个限定词准确？ | 只保留一个 |
| 通用结尾 | 未来一片光明、激动人心的时代即将到来 | 是否有确定下一步？ | 写下一步或直接结束 |
| 聊天残留 | 当然、希望这对您有帮助、请告诉我 | 是否是助手话术被粘进正文？ | 删除对话外壳 |
| 谄媚语气 | 好问题、您说得完全正确 | 是否只是在讨好？ | 直接回应相关观点 |

## C. 结构与节奏

这些项目不能靠单个词确定，必须阅读全文：

- 强行凑三个并列项；确有三个不可替代的信息时保留。
- 连续句子长度和结构相同；合并普通信息，让重点单独停顿。
- 全文一行一句；普通信息自然连说。
- 每段结尾重复总结；删除没有新增信息的复述。
- 每部分都有整齐标题和编号；口播改用真实问题或现场推进。
- 结论换词重复、同一对象循环使用同义词；保留最准确的称呼。
- 破折号制造反转或金句；优先改成正常标点。
- 比喻后再次解释；保留比喻或解释之一。
- 结尾突然上升到行业、时代和未来；回到具体行动或自然结束。
- 只有概念、能力和趋势，没有人、场景与动作；只能要求作者补充，不能代写事实。
- 全文没有判断或没有一次具体失败；仅在原文证据允许时恢复作者立场。
- 开头先介绍概念、产品或新闻；优先进入读者正在遇到的问题。

## 文体适配

- 口播：优先可张嘴说出的自然句子，允许正常重复，不把每句切成金句。
- 文章：保持段落逻辑，减少机械连接和段尾总结。
- 正式说明：保持中性和精确，不强行加入第一人称、幽默或情绪。
- 社交媒体：可以保留作者已有的锋芒，但不要添加夸张承诺。
